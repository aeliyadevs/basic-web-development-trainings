Introduction

This course is designed for absolute beginners who are interested in learning web development. This course aims to cover the fundamentals of developing and hosting a static website. Over the span of 3 months, students will gain fundamental knowledge in HTML, CSS, and JavaScript, the building blocks of the web.

Objectives

  1. Introduce students to the fundamentals of web development
  2. Guide students how to create and style web pages using HTML & CSS.
  3. Familiarize students with basic programming concepts through JavaScript.
  4. Provide hands-on practice and real-world projects to reinforce learning.

Course Duration

  • Duration: 3 Months
  • Frequency: Classes held every alternate day (1 hour class - 3 days a week)

Course Outline

  1. HTML

      • Lists (unordered, ordered and description lists)
      • Containers (div and span)
    • Week 2
      • Tables
      • Forms and input controls
      • Elements for embedding media (picture, video and audio)
      • Semantic HTML5 elements (main, section, article etc.)
      • HTML Validation
  2. CSS
    • Week 3
      • CSS introduction
      • Inline vs Internal vs External CSS
      • Selectors & properties
      • Essential CSS properties (color, padding, margin, border, shadow, etc)
      • Measurement units (px, %, em, rem, vw, vh, ch)
      • Positioning (absolute, relative, static, sticky, fixed)
    • Week 4
      • The Box Model
      • Overflowing
      • Selectors specificity
      • Flexbox & Grids
      • Responsiveness & Media Queries
    • Week 5
      • Mid-course Project 1
    • Week 6
      • Mid-course Project 1 continuation
